# 网络场景卡片（15个典型场景）
所有选项中，A选项均为最佳解决方案，B选项均为可接受方案，C选项为不当方案。在实际制作卡片时，请打乱选项顺序。

## A. 班级群聊场景（5个）

### 1. 班级大群
- 地点：班级微信群
- 事件：有人在群里发你的糗图
- 角色：班级群成员
- 选项：
  - A：截图保存，报告班主任
  - B：私聊对方请求删除
  - C：发对方的糗图报复

### 2. 学习小组群
- 地点：作业交流群
- 事件：有人嘲笑你的作业错误
- 角色：小组成员
- 选项：
  - A：保存聊天记录，告诉老师
  - B：退出该群，加入其他小组
  - C：在群里辱骂对方

### 3. 班委群
- 地点：班干部工作群
- 事件：有人在群里造谣说你偷懒
- 角色：其他班干部
- 选项：
  - A：用证据澄清并报告班主任
  - B：私下和对方沟通解释
  - C：在群里吵架对骂

### 4. 兴趣社团群
- 地点：社团活动群
- 事件：有人恶意P图损你名誉
- 角色：社团成员
- 选项：
  - A：收集证据，告诉指导老师
  - B：请群主删除不当图片
  - C：制作P图反击对方

### 5. 值日群
- 地点：值日安排群
- 事件：有人在群里孤立你
- 角色：值日小组成员
- 选项：
  - A：截图证据，找班主任帮助
  - B：专注完成自己的任务
  - C：拉帮结派对抗

## B. 社交平台场景（5个）

### 6. 朋友圈
- 地点：微信朋友圈
- 事件：有人在你的照片下恶意评论
- 角色：班级同学
- 选项：
  - A：保存评论，告诉家长老师
  - B：删除这条动态屏蔽对方
  - C：发帖辱骂对方

### 7. 视频平台
- 地点：抖音/快手
- 事件：有人把你的出糗视频上传
- 角色：同校学生
- 选项：
  - A：收集链接，报告老师处理
  - B：请求平台删除视频
  - C：上传对方的糗事

### 8. 直播间
- 地点：游戏直播间
- 事件：有人带节奏说你作弊
- 角色：网络玩伴
- 选项：
  - A：保存记录，告诉家长处理
  - B：关闭直播，退出游戏
  - C：在直播间互骂

### 9. 网络游戏
- 地点：多人在线游戏
- 事件：同学组团在游戏里欺负你
- 角色：班上玩家
- 选项：
  - A：截图证据，告诉老师
  - B：换个区服继续玩
  - C：找人组团报复

### 10. 社交评论区
- 地点：动态评论区
- 事件：有人带节奏说你坏话
- 角色：网络好友
- 选项：
  - A：保存评论，报告平台
  - B：关闭评论功能
  - C：组织网络攻击

## C. 即时通讯场景（5个）

### 11. 私聊对话
- 地点：私人聊天窗口
- 事件：收到陌生人威胁信息
- 角色：网络陌生人
- 选项：
  - A：不回复，告诉家长老师
  - B：立即拉黑对方
  - C：回复威胁对方

### 12. 语音通话
- 地点：网络语音
- 事件：有人录下你的语音乱剪辑
- 角色：群聊成员
- 选项：
  - A：收集证据，报告老师
  - B：减少语音聊天
  - C：也去录对方语音

### 13. 视频会议
- 地点：网课直播间
- 事件：有人截图你的尴尬表情
- 角色：班级同学
- 选项：
  - A：告诉任课老师处理
  - B：调整摄像头角度
  - C：也去截对方图片

### 14. 共享文档
- 地点：在线作业文档
- 事件：有人恶意修改你的作业
- 角色：同组同学
- 选项：
  - A：查看修改记录报告老师
  - B：及时备份自己的内容
  - C：去破坏对方作业

### 15. 班级投票
- 地点：投票小程序
- 事件：有人组织投票孤立你
- 角色：班级同学
- 选项：
  - A：截图证据告诉班主任
  - B：不参与相关投票
  - C：发起反对投票

## 网络场景设计特点

1. 安全措施：
   - 保存证据
   - 及时报告
   - 警惕陌生人

2. 预防要点：
   - 保护隐私
   - 谨慎分享
   - 定期备份

3. 处理原则：
   - 不随意转发
   - 避免对抗
   - 理性应对

4. 注意事项：
   - 关注时效性
   - 重视影响范围
   - 保护个人信息
