# 教室类场景卡片（20个典型场景）
所有选项中，A选项均为最佳解决方案，B选项均为可接受方案，C选项为不当方案。在实际制作卡片时，请打乱选项顺序。

## A. 普通教室场景（8个）

### 1. 课前准备
- 地点：普通教室前排
- 事件：有人把你的值日安排改成自己的名字
- 角色：值日小组长
- 选项：
  - A：请其他同学作证，向班主任报告
  - B：私下和小组长沟通
  - C：擅自改回自己名字

### 2. 课堂活动
- 地点：普通教室中间
- 事件：回答问题时有人在后面起哄模仿你
- 角色：后排同学
- 选项：
  - A：继续自信发言，课后告诉老师
  - B：暂时忍耐，专注回答问题
  - C：转身呵斥对方

### 3. 小组讨论
- 地点：普通教室分组区
- 事件：组员故意不给你发言机会
- 角色：同组同学
- 选项：
  - A：举手向老师反映分组问题
  - B：记录下来，找机会私下沟通
  - C：强行打断他人发言

### 4. 课间休息
- 地点：普通教室后门
- 事件：有人堵住门不让你出去
- 角色：其他班同学
- 选项：
  - A：大声请求帮助，叫班干部或老师
  - B：从另一个门离开
  - C：用力推开他们

### 5. 分组活动
- 地点：普通教室座位
- 事件：分组时大家互相推诿不愿和你一组
- 角色：同班同学们
- 选项：
  - A：冷静告诉老师，请求帮助合理分组
  - B：独自完成任务，认真做好自己
  - C：生气指责同学们势利

### 6. 考试期间
- 地点：普通教室考试座位
- 事件：有人故意踢你的椅子影响答题
- 角色：后座同学
- 选项：
  - A：举手告诉监考老师
  - B：往前挪动座位继续答题
  - C：回头警告对方

### 7. 值日清洁
- 地点：普通教室清洁区
- 事件：有人故意在你刚拖完的地上踩水印
- 角色：同班同学
- 选项：
  - A：记录下来，向值日老师反映
  - B：等对方走后再重新拖一遍
  - C：把脏水泼向对方

### 8. 放学整理
- 地点：普通教室储物柜
- 事件：发现有人在你的书本上乱涂乱画
- 角色：不知名同学
- 选项：
  - A：请其它同学作证，向班主任求助
  - B：用橡皮擦掉，保管好书本
  - C：去涂画别人的书

## B. 专业教室场景（7个）

### 9. 美术教室
- 地点：美术教室画架区
- 事件：有人恶意把颜料蹭到你的作品上
- 角色：邻座同学
- 选项：
  - A：请其它同学作证，告诉美术老师
  - B：尝试补救画作，换个座位
  - C：把颜料蹭回去

### 10. 音乐教室
- 地点：音乐教室表演台
- 事件：独唱时有人起哄嘲笑你的声音
- 角色：台下同学
- 选项：
  - A：继续完成表演，课后告诉老师
  - B：停下来等大家安静
  - C：对嘲笑的人大声反击

### 11. 科学实验室
- 地点：实验操作台
- 事件：有人偷换你的实验材料
- 角色：实验小组成员
- 选项：
  - A：立即向实验老师报告
  - B：仔细检查后重新准备
  - C：破坏对方的实验

### 12. 计算机教室
- 地点：电脑座位区
- 事件：有人趁你离开时改动你的程序
- 角色：同班同学
- 选项：
  - A：截图保存，报告信息老师
  - B：重新编写，注意保存
  - C：删除对方的程序文件

### 13. 舞蹈教室
- 地点：舞蹈教室镜子前
- 事件：有人模仿嘲笑你的舞姿
- 角色：舞蹈班同学
- 选项：
  - A：专注练习，告诉舞蹈老师
  - B：换个位置继续练习
  - C：出言讽刺对方

### 14. 阅览教室
- 地点：图书角
- 事件：有人撕破你正在看的图书
- 角色：高年级学生
- 选项：
  - A：保留证据，报告图书管理员
  - B：主动赔偿，换本书看
  - C：撕对方的书报复

### 15. 劳技教室
- 地点：工具区
- 事件：有人藏起你需要的工具
- 角色：同组同学
- 选项：
  - A：向劳技老师反映情况
  - B：借用其他组的工具
  - C：抢其他同学的工具

## C. 教师场景（5个）

### 16. 言语羞辱
- 地点：普通教室讲台前
- 事件：老师在全班同学面前因你考试成绩差而用刺激性语言批评你
- 角色：任课老师
- 选项：
  - A：保持冷静，课后找老师私下沟通或告诉班主任/家长寻求帮助
  - B：认真记笔记改正错误，暂时忍耐
  - C：顶撞老师或者逃课

### 17. 不当惩罚
- 地点：教室后排
- 事件：因为没交作业，老师让你整节课站着听讲
- 角色：任课老师
- 选项：
  - A：课后向班主任反映情况，请求合理的教育方式
  - B：默默接受惩罚，保证下次按时完成作业
  - C：拒绝执行并与老师争执

### 18. 区别对待
- 地点：教室前排
- 事件：老师总是忽视你举手回答问题，却经常让其他同学回答
- 角色：任课老师
- 选项：
  - A：找机会和班主任/家长反映，请他们帮助沟通
  - B：继续积极举手，用优秀表现证明自己
  - C：从此不再举手发言

### 19. 作业惩罚
- 地点：教室黑板前
- 事件：因做错题，老师要求你抄写同样的题目50遍
- 角色：任课老师
- 选项：
  - A：礼貌地表达这种惩罚可能影响其他功课的学习，请求更合理的改正方式
  - B：完成惩罚，同时认真理解题目要点
  - C：应付了事地完成或者干脆不写

### 20. 当众批评
- 地点：班级升旗仪式
- 事件：因为校服不整齐，老师在全年级同学面前点名批评你
- 角色：值周老师
- 选项：
  - A：主动找老师承认错误，并表达希望用其他方式提醒的想法
  - B：立即整理仪容仪表，保证以后注意
  - C：和周围同学抱怨老师小题大做

## 教室场景设计特点

1. 空间分布：
   - 前后排位置差异
   - 不同功能区域
   - 专业教室特点

2. 时间节点：
   - 课前准备
   - 上课期间
   - 课间活动
   - 放学时段

3. 互动类型：
   - 个人财物
   - 学习活动
   - 群体合作
   - 公共秩序

4. 解决策略：
   - 及时求助
   - 理性应对
   - 保留证据
   - 避免升级
